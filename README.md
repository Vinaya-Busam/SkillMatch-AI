# Task 5 - Resume Screening System (BNM Tech Solutions)

## Project Overview
**Resume Screening System** is a Machine Learning and NLP-based application designed to simplify and enhance the recruitment process for both recruiters and job seekers.

**For recruiters,** the application helps automate the initial screening process by evaluating resumes against job descriptions

**For candidates,** the system serves as a resume evaluation tool by providing an ATS match score that indicates how well their resume aligns with a specific job description.


**Dataset Source :** [Updated Resume Dataset](https://www.kaggle.com/datasets/jillanisofttech/updated-resume-dataset) 

**Check this out :** [Click here](https://resumeskillmatch.streamlit.app/)

## Features
- Resume Text Extraction
- Resume Classification using Machine Learning
- Skill Identification
- Job Description Matching
- ATS Match Score Generation
- Missing Skill Detection
- Interactive Streamlit Web Application
